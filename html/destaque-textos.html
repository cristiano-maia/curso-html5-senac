Destaque em Textos

Nesta aula, aprendemos a destacar informações importantes dentro de um texto.

O curso de HTML5 é a base para criar páginas na web.

Algumas palavras podem receber ênfase para chamar a atenção do leitor.

Também podemos usar marcações visuais para destacar partes importantes.

Exemplo de texto formatado

Meu nome é Seu Nome.
Estou estudando HTML5 no Senac.
Meu objetivo é aprender a criar páginas para a internet usando boas práticas de estruturação.
