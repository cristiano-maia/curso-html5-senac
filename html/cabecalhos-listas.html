Conteúdos estudados

  • Estrutura básica do HTML
  • Destaques em textos
  • Cabeçalhos
  • Listas ordenadas e não ordenadas
  • Links e imagens

Passos para criar uma página HTML

  1. Criar o arquivo com extensão .html
  2. Adicionar a estrutura básica do documento
  3. Inserir o conteúdo dentro da tag body
  4. Salvar o arquivo
  5. Abrir no navegador para testar
