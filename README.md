# Curso HTML5 - Senac

Este repositório reúne os arquivos desenvolvidos durante as aulas de HTML5.

O objetivo é organizar os conteúdos estudados em aula e praticar a criação de páginas HTML, links entre arquivos e publicação com GitHub Pages.

## Professor

Cristiano Maia

## Conteúdos estudados

Durante as aulas, foram praticados os seguintes conteúdos:

- Estrutura básica de uma página HTML
- Destaques em textos
- Cabeçalhos e listas
- Links e imagens
- Tabelas
- Formulários
